# 视觉适配器 (Visual Adapter) 技术报告

## 1. 概述

视觉适配器是针对 InCTRL 模型的视觉特征增强模块，通过引入轻量级的瓶颈残差 MLP 结构，在不破坏预训练 CLIP 视觉表示的前提下，对全局图像特征和局部 patch 特征进行任务适配，从而提升异常检测性能。

---

## 2. 原始 InCTRL 的适配器结构

### 2.1 Adapter 类

原始 InCTRL 使用一个简单的两层 MLP 作为适配器：

```python
# open_clip/model.py 第 430-442 行
class Adapter(nn.Module):
    def __init__(self, c_in, reduction=4):
        super(Adapter, self).__init__()
        self.fc = nn.Sequential(
            nn.Linear(c_in, c_in // reduction, bias=False),
            nn.ReLU(inplace=True),
            nn.Linear(c_in // reduction, c_in, bias=False),
            nn.ReLU(inplace=True)
        )

    def forward(self, x):
        x = self.fc(x)
        return x
```

**特点：**
- 瓶颈结构：输入 640 维 → 压缩至 160 维 → 恢复至 640 维
- 无残差连接：直接替换特征
- 无零初始化：MLP 输出会直接改变原始特征
- 仅处理全局特征：不涉及局部 patch 特征

### 2.2 原始 Adapter 在 forward 中的使用

```python
# open_clip/model.py 第 562-569 行（Visual Adapter 未启用时的分支）
token_ad = self.adapter.forward(token)           # 查询图像的全局特征
token_n = self.adapter.forward(token_n)          # 支持集正常图像的全局特征
token_n = torch.mean(token_n, dim=1)              # 对 shot 维度取平均
token_ref = token_n - token_ad                    # 计算特征差异
img_ref_score = self.diff_head_ref(token_ref)    # 异常分数（全局）
```

## 3. 视觉适配器 (Visual Adapter) 结构

### 3.1 ResMLP：瓶颈残差模块

```python
# open_clip/visual_adapter.py 第 9-34 行
class ResMLP(nn.Module):
    def __init__(self, c_in, reduction=4):
        super().__init__()
        self.fc = nn.Sequential(
            nn.Linear(c_in, c_in // reduction, bias=False),  # 压缩
            nn.BatchNorm1d(c_in // reduction),
            nn.ReLU(inplace=True),
            nn.Linear(c_in // reduction, c_in, bias=False), # 恢复
        )

    def forward(self, x):
        if x.dim() == 2:
            out = self.fc(x)
            return x + out  # 残差连接
        else:
            batch_size, seq_len, feature_dim = x.shape
            x_flat = x.reshape(batch_size * seq_len, feature_dim)
            out_flat = self.fc(x_flat)
            out = out_flat.reshape(batch_size, seq_len, feature_dim)
            return x + out  # 残差连接
```

**关键区别：**

| 特性 | 原始 Adapter | ResMLP |
|------|-------------|--------|
| 残差连接 | 无 | 有 |
| BatchNorm | 无 | 有（压缩层后） |
| 激活函数位置 | 两端均有 ReLU | 仅中间有 ReLU |
| 输出初始状态 | 随机 | 近似恒等映射（zero_init） |

### 3.2 VisualAdapter 双路结构

```python
# open_clip/visual_adapter.py 第 37-95 行
class VisualAdapter(nn.Module):
    def __init__(
        self,
        img_size: int,
        patch_size: int,
        global_input_dim: int,   # embed_dim = 640
        local_input_dim: int,    # vision_cfg.width = 768
        reduction: int = 4,
        zero_init: bool = True,
    ):
        super().__init__()
        self.global_adapter = ResMLP(global_input_dim, reduction=reduction)  # 用于全局特征
        self.local_adapter = ResMLP(local_input_dim, reduction=reduction)    # 用于 patch 特征

        if zero_init:
            self._zero_init_last_layer()

    def _zero_init_last_layer(self):
        """零初始化最后一层，初始状态等价于恒等映射"""
        self.global_adapter.fc[-1].weight.data.zero_()
        self.local_adapter.fc[-1].weight.data.zero_()

    def adapt_global(self, x: torch.Tensor) -> torch.Tensor:
        """处理 [B, dim] 或 [B, S, dim] 形状的全局特征"""
        return self.global_adapter(x)

    def adapt_local(self, patch_tokens: list) -> list:
        """处理多层 patch 特征列表，每个元素 [B, N_patches, dim]"""
        adapted = []
        for patch_feat in patch_tokens:
            adapted.append(self.local_adapter(patch_feat))
        return adapted
```

## 4. 视觉适配器在 InCTRL forward 中的集成

### 4.1 全局特征适配

```python
# open_clip/model.py 第 543-549 行
if self.visual_adapter is not None:
    # 全局特征适配
    token_ad = self.visual_adapter.adapt_global(token)      # [B, 640]
    token_n = self.visual_adapter.adapt_global(token_n)      # [B, shot, 640]
    token_n = torch.mean(token_n, dim=1)                    # 对 shot 维度取平均
    token_ref = token_n - token_ad                          # 异常特征差异
    img_ref_score = self.diff_head_ref(token_ref)           # 全局异常分数
```

### 4.2 局部特征适配

```python
# open_clip/model.py 第 551-561 行
    # 将 4D patch 特征转换为 list of 3D
    # Fp_list: [3, B, 225, dim] → [B, 225, dim] per layer
    Fp_list_3d = [Fp_list[:, layer_idx, :, :] for layer_idx in range(Fp_list.shape[1])]
    adapted_Fp_list = self.visual_adapter.adapt_local(Fp_list_3d)
    adapted_Fp_list = torch.stack(adapted_Fp_list, dim=1)   # 恢复为 [B, 3, 225, dim]

    # 同样处理 normal 样本的 patch 特征
    # Fp_list_n: [B, 3, 225*shot, dim] → [B, 225*shot, dim] per layer
    Fp_list_n_3d = [Fp_list_n[:, layer_idx, :, :] for layer_idx in range(Fp_list_n.shape[1])]
    adapted_Fp_list_n = self.visual_adapter.adapt_local(Fp_list_n_3d)
    adapted_Fp_list_n = torch.stack(adapted_Fp_list_n, dim=1)
```

### 4.3 适配后 patch 级别的异常计算

```python
# open_clip/model.py 第 581-606 行
# 适配后的 patch 特征用于计算 patch 级别的相似度
for j in range(n_layers):
    tmp_x = adapted_Fp_list[:, j, :, :]        # 查询图像的 patch 特征 [B, 225, dim]
    tmp_n = adapted_Fp_list_n[:, j, :, :]      # 正常图像的 patch 特征 [B, 225*shot, dim]

    # 归一化
    tmp_x_norm = tmp_x / tmp_x.norm(dim=-1, keepdim=True)
    tmp_n_norm = tmp_n / tmp_n.norm(dim=-1, keepdim=True)

    # 计算余弦相似度矩阵
    sim = 0.5 * (1 - torch.bmm(tmp_x_norm, tmp_n_norm.transpose(-2, -1)))

    # 每个 patch 与所有正常 patch 的最小相似度
    min_dist = sim.min(dim=-1).values
    Fp_map_per_layer.append(min_dist)

# 多层平均得到最终 patch 异常图
Fp_map = torch.stack(Fp_map_per_layer, dim=0).mean(dim=0)
patch_ref_map = Fp_map  # [B, 225]
max_diff_score = Fp_map.max(dim=1).values  # [B]
```

## 5. 性能提升分析

### 5.1 VisA 数据集测试结果

基于 `inctrl_va_bs48_lr0.001_ep10_epoch_10` 的测试结果：

| Shot | 2-shot | 4-shot | 8-shot |
|------|--------|--------|--------|
| AUROC | 0.8747 | 0.8903 | 0.9010 |
| AUPR | 0.8827 | 0.8956 | 0.9051 |

### 5.2 提升原因分析

#### 5.2.1 残差连接保留预训练知识

原始 Adapter 无残差连接，其输出直接替换原始特征。在训练早期，随机初始化的 MLP 会破坏 CLIP 的预训练表示，导致模型需要较长时间恢复。

Visual Adapter 的残差结构 `output = input + MLP(input)` 保证：
- 初始状态近似恒等映射：`output ≈ input`
- CLIP 预训练特征始终保持可用
- 适配器逐步学习"修正量"而非"新表示"

#### 5.2.2 零初始化确保稳定起步

```python
def _zero_init_last_layer(self):
    self.global_adapter.fc[-1].weight.data.zero_()
    self.local_adapter.fc[-1].weight.data.zero_()
```

由于最后一层权重为零：
- 训练开始时：`MLP(x) ≈ 0`，`output = input + 0 = input`
- 模型行为完全等价于原始 InCTRL
- 随着训练进行，适配器逐步学习有益的特征修正

这对 few-shot 场景尤为重要——小样本下学习的稳定性至关重要。

#### 5.2.3 BatchNorm 提升特征稳定性

ResMLP 在压缩层后添加了 BatchNorm：
- 压缩后的特征分布更稳定
- ReLU 激活更有效
- 对输入变化更鲁棒

#### 5.2.4 局部 patch 特征的独立适配

原始 InCTRL 只有 Adapter 处理全局特征，patch 级别的相似度计算直接使用原始 CLIP 特征。Visual Adapter 增加了 `local_adapter`：

```
全局特征路径：token → global_adapter → token_ad
局部特征路径：Fp_list → local_adapter → adapted_Fp_list
```

这使得：
- 全局语义异常（整体缺陷）被更好地捕捉
- 局部纹理异常（表面划痕、污点）被更好地识别
- patch 级别的相似度计算基于适配后的特征，更适合异常检测任务

#### 5.2.5 随 shot 数增加提升更大的原因

从结果看，shot 数越多提升越明显：
- 2-shot → 8-shot 期间 AUROC 从 0.8747 提升到 0.9010
- 更多 normal samples 使适配器学习到更准确的"正常"分布边界
- 局部特征在更多 shot 下统计更稳定

## 6. 架构对比总结

| 特性 | 原始 InCTRL (Adapter) | InCTRL+VA (VisualAdapter) |
|------|---------------------|--------------------------|
| 残差连接 | 无 | 有 |
| BatchNorm | 无 | 有（压缩层后） |
| 激活函数 | 两端 ReLU | 仅中间 ReLU |
| 处理特征类型 | 仅全局特征 | 全局 + 局部 patch |
| 零初始化 | 无 | 有 |
| 初始状态 | 随机破坏特征 | 恒等映射（保持 CLIP） |
| patch 特征适配 | 直接使用 CLIP 特征 | 通过 local_adapter 适配 |
| 可学习参数 | 较少 | 略多（但仍轻量） |

## 7. 关键技术点

### 7.1 瓶颈设计降低计算开销

```
输入 dim=640 → Linear(640→160) → BN → ReLU → Linear(160→640) → 输出
```

 reduction=4 使得参数量从 `640×640 = 409,600` 降至 `640×160 + 160×640 = 204,800`，减少 50%。

### 7.2 支持 2D 和 3D 输入

```python
def forward(self, x):
    if x.dim() == 2:  # [B, dim] 全局特征
        out = self.fc(x)
        return x + out
    else:             # [B, S, dim] 或 [B, N, dim] patch 特征
        batch_size, seq_len, feature_dim = x.shape
        x_flat = x.reshape(batch_size * seq_len, feature_dim)
        out_flat = self.fc(x_flat)
        out = out_flat.reshape(batch_size, seq_len, feature_dim)
        return x + out
```

### 7.3 多层 patch 特征的独立适配

Visual Adapter 的 `adapt_local` 方法接收多层 patch 特征列表，分别通过同一个 `local_adapter` 进行适配，保证多层特征的一致性处理。

---

## 8. 结论

视觉适配器通过引入瓶颈残差结构，在保持 CLIP 预训练知识完整性的同时，实现了对异常检测任务的有效特征适配。零初始化、BatchNorm 和双路（全局+局部）设计使其在 few-shot 场景下表现出色，显著优于原始 InCTRL 的简单 MLP 适配方案。
